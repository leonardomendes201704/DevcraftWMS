# agent.md - Views (Portaria)

## Purpose
- Razor views for the Customer Portaria UI.

## Allowed dependencies
- ViewModels, shared partials, Bootstrap 5.

## Mandatory patterns
- All form inputs use input groups with left icons.
- Index screens include Help button and manual content.
- Use shared partials for alerts and layout.

## Forbidden actions
- Business logic in views.
- Direct API calls from views.

## Acceptance checklist
- EN-US labels.
- Friendly empty states and errors.
- Help/manual updated when screens change.
